**npm**

```bash
npx prisma
```

**Analyze the database and write its schema to the `schema.prisma` file**

```bash
npx prisma db pull
```

**Push the Schema**

```bash
npx prisma push
```

**Push the schema, accepting data loss:**

```bash
npx prisma db push --accept-data-loss
```

**Prisma studio**

```bash
npx prisma studio
```
